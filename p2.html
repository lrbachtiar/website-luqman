Creating a Static Resume Website

Around middle of 2015 I setup a personal website using wordpress hosting. Just last month I received an email reminder to renew these hosting services. During the first year of service, bluehost provided a special offer of USD$3.49/month wordpress hosting. However, since I am entering the second year of hosting, these costs would increase to the normal minimum rate of $7.99/month. For me personally, this website isn't anything too important and definitely not worth the annual $100 cost. So I had a look around the interwebs and found something interesting! It turns out github allows you to create basic static pages for free: link. Github provides a basic overview of how to use this feature, so in this post I'll just outline the additional things I found when setting things up.

Things you need:

  1. Knowledge of Git and a Github account
  2. Basic HTML knowledge and a HTML website template.
  3. Purchased domain name.

Using Github

To host a static for free on github, you must at least know some basic usage of git. Git is a code management system and for our purposes, it is used to upload the html contents of the website into the github repository.

HTML website template

I have little to-no experience with HTML so I'm completely fine using templates. To be honest, using templates is a great way to understand the mechanics of HTML and how different elements are used. I initially started with the skeleton template (link).

In addition to my inexperience with HTML, I'm also horrible at design. I started with the skeleton template and tried to at least make a decent looking website but failed miserably. I found something called bootstrap (link), some kind of HTML framework that's quite popular, which means many pre-made free templates to choose from. Great example of a website made with bootstrap: Seriously, Open This Page, It's Awesome

Using your purchased domain name

Once you have completed your website locally and committed it into your gh-pages branch, you can now link a purchased domain name to it. The original address of my website is actually: http://lrbachtiar.github.io/website-luqman. I've linked my domain name luqmanbachtiar.com by adding a CNAME file into the gh-pages branch. You will then need to use your DNS manager to the CNAME records to the github ip address (link).

That's all there is to it folks :)
